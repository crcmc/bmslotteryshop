  🏝️ 보물섬 복권방 - 사용 방법
═══════════════════════════════════════════════════

[ 즉시 실행 ]
  ▶ start.bat 더블클릭
     - Python이 설치돼 있으면 로컬 서버(http://localhost:8765)로 열려
       동행복권 API가 정상 동작합니다.
     - Python이 없으면 기본 브라우저로 직접 열립니다.

[ 부팅 시 자동 실행 등록 ]
  ▶ install-autostart.bat 더블클릭
     - Windows 시작 폴더에 바로가기를 등록합니다.
     - 다음 PC 부팅(로그인) 시 자동으로 복권방이 열립니다.

[ 자동 실행 해제 ]
  ▶ uninstall-autostart.bat 더블클릭

[ 브라우저 시작 페이지로 설정하는 방법 (대안) ]
  Chrome / Edge:
   1) 설정 → "시작 그룹" → "특정 페이지 또는 페이지 모음 열기"
   2) 새 페이지 추가 → 아래 주소 입력
      file:///C:/MC/bmslotteryshop/index.html
      또는 (start.bat 실행 후)
      http://localhost:8765/index.html

  주요 기능
═══════════════════════════════════════════════════
  1. 🎲 로또 번호 자동 생성

  3. 🏆 지난 회차 당첨번호 조회
     - 동행복권 공식 API 직접 조회
     - 최신 회차 자동 탐지, 회차별 직접 조회 가능
     - 최근 5회차 한눈에 보기
